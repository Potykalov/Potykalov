### 🪢 [Игра «Виселица»](https://github.com/Potykalov/hangmanGame)
[Демо](https://potykalov.github.io/hangmanGame/)

Классическая игра «Виселица» на JavaScript, в которой игрок угадывает скрытое слово, выбирая буквы с экранной клавиатуры.

**Что реализовано:**
- запуск новой игры со сбросом состояния;
- выбор случайного слова из словаря;
- обработка кликов по виртуальной клавиатуре;
- проверка правильных и ошибочных букв;
- обновление состояния слова, клавиатуры и игрового персонажа;
- проверка победы и поражения.
  
**Стек:** `JavaScript` `DOM API` `Git`

**Вывод:** 
Закрепил работу с состоянием приложения, строками, массивами, условиями, функциями, событиями и обновлением интерфейса через DOM.

### 📱 [Адаптивная и мобильная вёрстка](https://github.com/Potykalov/mq-diploma)
[Демо](https://potykalov.github.io/mq-diploma/)

Адаптивная страница, свёрстанная по макетам для десктопных экранов, планшетов и мобильных устройств.

**Что реализовано:**
- адаптивная вёрстка страницы по PSD-макетам;
- корректное отображение интерфейса на разных ширинах экрана;
- резиновая вёрстка для промежуточных состояний между основными макетами;
- адаптация блока «Сейчас в тренде» под разные размеры и ориентации экрана;
- всплывающая форма;
- бургер-меню;
- корректное отображение контента и изображений при изменении размеров экрана.

**Стек:** `HTML` `CSS` `Flexbox` `Media Queries` `Git`

**Вывод:**
Закрепил работу с адаптивной и резиновой вёрсткой, media queries, Flexbox, относительными размерами и адаптацией изображений. Получил практику настройки интерфейса для разных разрешений и ориентаций экрана.
